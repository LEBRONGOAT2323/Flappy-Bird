# ЁЯза How It Works

Gravity constantly pulls the bird down.

Press jump to apply upward velocity.

Pipes spawn at intervals with randomized gaps.

Score increases each time you pass a pipe.

Collision detection checks:

Top / bottom of screen

Pipe rectangles

Camera shake is applied briefly when jumping to make the movement feel more responsive and impactful.
